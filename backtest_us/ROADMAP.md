# NASDAQ 퀀트 — 로드맵 & 방향

> 벤치마크 철학: 김민겸(2025 WorldQuant IQC 단독우승)
> 소수정예 · 안정적 우상향(Sharpe/MDD) · 레짐 분산 · AI=조교

## 확정된 방향 (2026-06)
- **자동화 = L1 (반자동)**: 시스템이 주 1회 바스켓 제안 → **사람 승인** → KIS 집행.
  AI는 복기·리포트만, 매매 결정은 사람.
- **신·구 관계 = 수렴**: 검증 끝나면 `kis/` 라이브 봇의 즉흥 진입규칙을
  `backtest_us`에서 **검증된 시그널로 교체**. 봇은 "집행기"로만 남김.
- **시장 = 미장(NASDAQ) 집중**: 첫 알파는 미장 한 곳. (한국장은 이후)

## 합격 기준 (돌리기 전에 못박는다 — 결과에 끌려다니지 않기)
첫 알파(Clenow 미장)는 아래를 **모두** 만족해야 채택:
1. **QQQ 매수후보유 대비 Sharpe 더 높음**
2. **QQQ 대비 MDD 더 얕음** (또는 비슷한데 Sharpe 우위)
3. 파라미터(lookback/top_n)를 흔들어도 성과 유지 (특정값 과적합 아님)

하나라도 못 넘으면 → 알파 수정 or 폐기. **PnL 크기는 판단 기준 아님.**

## 작업 큐
```
[1] 검증      backtest_us 로 Clenow 미장 백테스트            ← 현재 단계
              실행: GitHub Actions(폰) 또는 맥 로컬
[2] 견고성    walk-forward + lookback/top_n 민감도 점검
[3] 시그널화  검증된 알파 → 주1회 "바스켓+비중" JSON
              여기서 kis/ 즉흥 진입규칙 교체
[4] L1 집행   텔레그램 제안 → 사람 승인 → KIS 주문
[5] 조교 루프 intelligence/ 가 체결·저널·복기 리포트
[6] 2번째 알파 리버전 sleeve 추가 (레짐 분산) — [1] 합격 후에만
```

## 폰 ↔ 맥 ↔ Actions 분업
| 환경 | 네트워크 | 역할 |
|------|---------|------|
| **폰 (Claude on web)** | GitHub/PyPI만 | 코드 작성·엔진 검증(합성)·커밋·**Actions 트리거** |
| **GitHub Actions 러너** | 열림 | stooq 데이터 받아 **실제 백테스트** → 결과 커밋 + 텔레그램 |
| **맥 (로컬, 선택)** | 열림 | 대량 실험·탐색 시 |

→ `.github/workflows/nasdaq_backtest.yml` 로 **폰만으로 전체 루프 가능**.
   맥은 있으면 편하지만 필수 아님.

## 현재 상태
- ✅ `backtest_us/` 횡단면 백테스터 — 무미래참조 검증 통과(자본곡선 차이 0.00)
- ✅ [1] 실제 데이터 백테스트 (GitHub Actions, yfinance) — 잠정 합격 후 생존편향에서 뒤집힘
- ✅ [2] 견고성 — 생존편향 정면돌파 **완료**. 결론: 종목선택 알파 없음, 레짐필터만 견고.
- 🔜 다음: **레짐필터를 알파로 승격**(A안) — 종목선택 모멘텀은 폐기/보류.

### [2] 생존편향 정면돌파 — EW유니버스 통제 (2026-06-03, Actions, run #4)
사전 합의 판정규칙: "EW유니버스(같은 승자 동일가중 보유)보다 비슷/열위면 알파 재설계."

| 지표 | Clenow top12 | QQQ B&H | **EW-universe** | 판정 |
|------|---:|---:|---:|:--:|
| Sharpe | 0.79 | 0.52 | **0.92** | ❌ EW 승 |
| Sortino | 0.88 | 0.69 | **1.23** | ❌ EW 승 |
| CAGR | 13.5% | 11.0% | **19.7%** | ❌ EW 승 |
| Calmar | 0.38 | 0.13 | 0.39 | ≈ 무승부 |
| MDD | **−35.6%** | −83.0% | −49.9% | ✅ Clenow 승 |

**결론:**
- 1차 "잠정 합격"(QQQ 대비 우월)은 **대부분 생존편향**. EW유니버스도 QQQ를
  동일하게 압도(Sharpe 0.92) → 그 격차가 곧 생존편향 크기.
- Clenow **종목선택 알파 = 없음(오히려 EW에 열위)**. 유일한 진짜 엣지는
  **레짐필터의 MDD 통제**(−35.6% vs EW −49.9% vs QQQ −83%, 73%만 투자).
- ⚠️ **공정성 한계**: 승자-only 유니버스는 구조적으로 모멘텀에 불리(죽은
  종목 0 → 모멘텀의 "패자 회피" 강점을 측정 불가). 상장폐지 데이터 없이는
  종목선택 알파를 공정하게 검증 불가능.

### [2b] 상장폐지 데이터 확보 → PIT 공정검증 (2026-06-03, pit_backtest run #2)
사용자 결정 **B** 실행. fja05680/sp500 시점별 멤버십 + stooq/yfinance 가격.
유니버스 766종목(상폐 272 포함), 1993–2026, SPY 벤치마크.

| 지표 | Clenow top12 | SPY B&H | **EW-universe** | 판정 |
|------|---:|---:|---:|:--:|
| Sharpe | 0.62 | 0.65 | **0.71** | ❌ Clenow 꼴찌 |
| Sortino | 0.70 | 0.83 | **0.86** | ❌ Clenow 꼴찌 |
| CAGR | 9.2% | 10.9% | **12.3%** | ❌ Clenow 꼴찌 |
| MDD | **−25.8%** | −55.2% | −56.1% | ✅ Clenow 압승 |
| Calmar | **0.36** | 0.20 | 0.22 | ✅ Clenow 압승 |

**최종 결론 (두 유니버스 일관):**
- **종목선택 모멘텀 알파 = 없음.** survivor·PIT 양쪽 모두 EW-universe가
  Sharpe/Sortino/CAGR 우위. 모멘텀 선택 < 단순 동일가중 보유.
- **유일하게 견고·재현되는 엣지 = 레짐필터의 낙폭 통제**(MDD −25.8% vs −56%,
  Calmar 0.36). 종목선택과 무관, 두 유니버스 모두에서 확인 → **이것이 진짜 알파**.
- 한계: PIT 커버리지 64%(상폐 691중 272만, stooq 레이트리밋). 절대수익은
  잔여 생존편향으로 다소 과대. 단 Clenow vs EW는 동일 유니버스라 상대판정 견고.
- 데이터 정제: sanitize()로 미조정-split/심볼재사용 57종목 드롭(BEN 등 일부
  정상주 오탈락 비용 감수). stooq 미조정이 무료 상폐데이터의 근본 한계.

### 채택 방향: A — 레짐필터를 알파로 승격
