# DETAILAB — 상세페이지 디자인 회사 홈페이지 (프로토타입)

매출을 만드는 상세페이지 디자인 스튜디오를 위한 모션 중심 원페이지 랜딩입니다.
순수 HTML/CSS/JS로 제작되어 **외부 라이브러리 없이** 브라우저에서 바로 미리보기할 수 있습니다.

## 구성 섹션
1. **히어로** — 애니메이션 그라데이션 배경, 텍스트 라이즈 모션, 마우스 패럴랙스 목업, 숫자 카운트업
2. **클라이언트 마키** — 무한 흐름 로고 띠
3. **포트폴리오** — 카테고리 필터 + 호버 인터랙션 + 스크롤 리빌
4. **서비스** — 호버 글로우 카드
5. **가격** — 3단계 패키지 (PRO 강조)
6. **제작 과정** — 스크롤 진행에 따라 채워지는 타임라인
7. **CTA / 푸터** — 문의 유도

## 미리보기
```bash
cd website
python3 -m http.server 8080
# 브라우저에서 http://localhost:8080 접속
```
또는 `index.html`을 브라우저로 바로 열어도 됩니다.

## 커스터마이징
- **색상 / 둥근 정도 / 간격**: `css/style.css` 상단의 `:root` 변수만 바꾸면 전체 테마가 변경됩니다.
- **회사명·문구·가격**: `index.html`의 텍스트를 직접 수정하세요. (현재 브랜드명 `DETAILAB`은 예시입니다.)
- **포트폴리오 이미지**: 현재는 그라데이션 플레이스홀더입니다. `.card__thumb`에 실제 이미지를 넣으면 됩니다.

## 워드프레스로 옮기기
이 프로토타입은 워드프레스 이식을 전제로 만들었습니다. 두 가지 경로가 있습니다.

1. **커스텀 테마로 변환** — `index.html`을 `front-page.php`로, `css/`·`js/`를 테마 폴더에 넣고
   `functions.php`에서 `wp_enqueue_style/script`로 로드. (원하시면 테마 패키지로 만들어 드립니다.)
2. **빌더(Elementor 등)에 이식** — 섹션 단위 구조와 문구가 그대로라 블록으로 옮기기 쉽습니다.

> 다음 단계로 워드프레스 테마 패키지(`style.css` 헤더 + `functions.php` + 템플릿)가 필요하면 말씀해 주세요.
